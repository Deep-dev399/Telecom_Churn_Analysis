# GitHub
Customer Churn refers to the number of existing customers who may leave the service provider over  agiven period
1. Project Title :
Telecom Churn Analysis
2. Project Description :
This project focuses on analyzing customer churn in the telecom industry. The goal is to identify the factors that contribute to customer churn and build predictive models to help telecom companies retain customer.
3. Dataset :
The dataest I have taken is from a kaggle challenge.
4.Multiple models were implemented to predict customer churn, including:
- Logistic Regression
- Decision Trees
- Random Forest
- Gradient Boosting
